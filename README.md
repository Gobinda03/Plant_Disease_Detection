# 🌿 Plant Disease Detection System

An AI-powered web application that detects plant leaf diseases using Deep Learning and Computer Vision.

Built with **TensorFlow**, **CNN**, and **Streamlit**.

---

## 🚀 Live Demo

https://plant-disease-detection-gh.streamlit.app/

---

## 📌 Features

✅ Upload plant leaf image  
✅ Detect disease instantly  
✅ Confidence score  
✅ Supports 38 plant disease classes  
✅ Clean and simple UI  
✅ Fast predictions using CNN model

---

## 🧠 Tech Stack

- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pillow
